### Projeto de estudo em java simulando a Mega-Sena.

O sistema é um simulado de jogos da mega-sena. Por ele gera-se concursos, adiciona participantes, cria apostas para os participantes podendo ser aleatória ou números escolhidos,
e ao fim mostra-se o resultado do sorteio verificando se houve ganhadores.

### Os requisitos para o projeto são: 
- Deve-se implementar uma solução orientada a objetos para a Mega-Sena;
- Esta solução deve permitir que sejam criados sorteios, adicionados participantes, sorteados números por participante, bem como verificar se alguém ganhou;
- O prêmio é equivalente à soma do valor pago por cada participante (R$ 6,00 por jogo);
- Só podem ser feitos jogos com 6 números. Utilize os princípios SOLID para projetar e implementar uma boa solução.
